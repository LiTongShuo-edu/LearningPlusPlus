# 参考边界与原创声明

## 早期代码背景

本人于 2019 至 2020 年在旧电脑上编写过一批 C++ 教学练习代码。其中部分代码曾上传到个人 OneDrive 网盘并得以保留，部分代码随着旧电脑报废而遗失。本仓库使用无后缀目录归档仍保留的早期手写源码，并使用 `＊` 目录标识在 Codex 协助下重新复现的遗失练习代码。

## 公开内容范围

本仓库公开发布的内容限于：

- 本人早期手写 C++ 练习的源码副本，发布时仅进行显示所需的 UTF-8 编码转换和目录整理。
- 本人丢失的部分练习代码，在 Codex 协助下重新复现的 `＊` 案例。
- 面向测试开发岗位自行设计的 `^` 扩展示例。
- 围绕上述源码撰写的 README、学习路线和验证文档。

## 不公开内容

私人学习资料不作为本仓库公开成果发布。仓库不上传或转发课程讲义、课程图片、压缩包、录制材料、课程成品源码、可执行文件或其他第三方原件。

未取得第三方资料的公开再发布许可。教育学习目的不自动产生公开传播第三方内容的权利，因此 `＊` 复现代码不采用复制第三方资料源码后改名发布的方式。

## AI 协作标识

- 名称无后缀的源码为早期手写练习，不声称其题目设计或学习路径完全原创。
- 名称末尾为 `＊` 的源码对应本人丢失的部分练习代码，是在 Codex 协助下重新复现、调试和整理的公开版本。
- 名称末尾为 `^` 的源码是在 Codex 协助下为岗位匹配展示额外设计的公开代码。
- 文档同样经过 Codex 协助整理，并以可核实的构建与运行结果为表述边界。
